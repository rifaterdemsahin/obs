# Projector Hotkeys Plugin

An OBS Lua script that adds hotkey support for opening fullscreen projectors.

  • Hotkey-based projectors — Bind keys to open fullscreen projectors for Program output, Multiview, and individual scenes
  • Monitor selection — Choose which monitor (1–10) each projector opens on
  • Startup projectors — Automatically open projectors when OBS launches
  • Grey-screen fix — Optional duplicate-open workaround for the blank projector issue

Installation

  1. Open OBS Studio → Tools → Scripts
  2. Click + and select plugins/projector-hotkeys.lua
  3. Configure monitor assignments in the script properties panel
  4. Set hotkeys in Settings → Hotkeys

# Configuration Backup

The backup-obs-config.ps1 script copies your OBS configuration from %APPDATA%\obs-studio into obs-config-backup/.

# Default backup
PS> .\backup-obs-config.ps1

# Custom backup directory
PS> .\backup-obs-config.ps1 -BackupDir "C:\my-backups\obs"
